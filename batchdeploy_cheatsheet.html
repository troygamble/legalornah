ğŸ”„ 5. Automating Daily Deployments (GitHub Actions)

Create a GitHub Action for automated daily deployment:

mkdir -p .github/workflows
nano .github/workflows/scheduled-deploy.yml

Add this content:

name: Scheduled Deploy
on:
  schedule:
    - cron: '0 0 * * *'  # Every day at midnight
  workflow_dispatch:
jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
    - uses: actions/checkout@v3
    - name: Setup Ruby
      uses: ruby/setup-ruby@v1
      with:
        ruby-version: 3.3
    - name: Install dependencies
      run: bundle install
    - name: Jekyll Build
      run: bundle exec jekyll build
    - name: Commit and push
      run: |
        git config --global user.name 'GitHub Actions'
        git config --global user.email 'actions@github.com'
        git add .
        git commit -m "Automated daily deploy $(date)"
        git push origin staged-deploy
